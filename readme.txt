== Description ==

This plugin is to embed opulent Website Chatbot into wordpress site.


= Get Started =
You can get Website Chatbot quick, and easy. [Get Website Chatbot from our site](https://opulent.com) in minutes.

== Installation ==

= Automated Installation =
Install this wordpress plugin, get your API Key from Opulent and activate your Opulent Bot on your Website

= Manual Alternatives =
Get embed Code from your Bot Dashboard on Opulent, and put that code before &lt;/body&gt; on any page where you want to enable bot
